Use this tutorial to learn how to make a simple preloader. Note that this is best for small sized flash files, like those having loops or high resolution images. A tutorial on Percentage Preloaders is also available on this site.

Step 1 : Create a new document in Flash and on the first layer, select Frame 5, right click and select 'Insert Keyframe' or press 'F6'. Go to the first frame and type some text there using the text tool (T), like Loading or something.


Step 2 : Now select Frame 5 and open up the Actions panel. If it is not visible, you can get it displayed by selecting 'Window -> Actions' or pressing 'F9'. Type the following ActionScript into the Actions panel.


What this code does is checks for when the frames loaded becomes equal to the total number of frames in the movie. If they are not equal, then goback to play from 1st frame. If they are equal, do nothing or continue playing the movie.

What more can be done :

  • You can put your content after the 5th frame.
  • Put 'Stop();' on the last frame of the movie if you do not want the movie to show the loading message again.
  • You can put some animation or animation movieclips with the loading message. It would look cool.
