Circular Text

Application
Fireworks MX 2004 +
Difficulty
Beginner
Date Added
25 Aug 2006
AutoRating(out of 5)
***

This Tutorial is a continuation of curved text. It uses the same command, but the technique is slightly different.

Step 1 : Create a new document taking any value for height and width for the stage. Draw a circle using the Ellipse Tool (U). Now type the text that you want to make circular.


Step 2 : Select both the items and go to 'Text -> Attatch to Path'

Note : The best thing about this is that we can still edit the text when it is made circular.
