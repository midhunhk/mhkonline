Image Rollover using Only Css

Application
Notepad to Dreamweaver
Difficulty
Intermediate
Date Added
16 Aug 2006
AutoRating(out of 5)
****

     We are going to make this.

    First of all, you have to create two images of same dimensions for the rollover and normal states of the buttons. You can choose any size, color or program to create it, just that they should both be having same dimensions. We chose two images of size 95 X 95 with a black background. You can use the images that we use here. Normal Over Right click and press 'Save Target As'.

This is the CSS code. Paste it in the <HEAD> section between <STYLE type="text/css"> & </STYLE> tags.

.nav a{
display:block;
background-image:url(nav_norm.jpg);
background-repeat:no-repeat;
border:0px;
clear:both;
margin-bottom:0px;
margin-left:2px;
margin-right:2px;
text-align:center;
width:95px;
height:95px;
}
.nav a:hover{
display:inherit;
background-image:url(nav_over.jpg);
background-repeat:inherit;
}

This type of links are usually used in a navigation bar. So create a table because, the link text has to be almost the same size as the image. Also if you want to use a number of such links horizontally, a table is more suitable. The table code should look like this.

<table width="100" border="0" bgcolor="#000000">
<tr>
<td class="nav"><a href="#"><br /><br />Link 1 <br /></a></td>
<td class="nav"><a href="#"><br /><br />Link 2 <br /></a></td>
<td class="nav"><a href="#"><br /><br />Link 3 <br /></a></td>
<td class="nav"><a href="#"><br /><br />Link 4 <br /></a></td>
<td class="nav"><a href="#"><br /><br />Link 5 <br /></a></td>
<td class="nav"><a href="#"><br /><br />Link 6 <br /></a></td>
</tr>
</table>

That is it. no messy javascript, and with just two images you can create any number of rollovers.
