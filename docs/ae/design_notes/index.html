Redesigning Mhkonline

This section is all about the design challenges that we faced during the development of this site. We have included and updating it for completeness. The redesign phenomenon started early in April of 2006. There were a lot of issues with time management so the relaunch was delayed a number of times. But now all developmental activities are at a full swing. If you are a budding web developer, you can ask for whatever tips or doubts you have and we will try to solve it.

Design Challenges

The entire idea of developing this site revolves around the fact that each section is like a mini site. i.e, each section has its own themes and styles, yet they all use some common standard properties. So when it comes to style, we used a global CSS file that is used by all the web pages in this site. Most of the sections have their of style sheets to supplement this and to store their own style. We have used to float images with CSS at a number of places so that the image seamlessly integrates with text.

There were a number of design related challeges that we had to overcome. This site relies extensively on CSS based styles. The Tutorials section requires Javascript as we used Javascript to act as a database of all the tutorials and other related information. We will be concentrating on more tutorials once we finish the other works on this site.

We have tried to make each section unique with their own styles and themes, like ministes. But all of the pages in this site have some inherent common properties. Perhaps the most important part of any site would be the navigation. So we spent a great deal of time in perfecting the navigation scheme of each section. Simple and Effective!
