Welcome to slideviewer at mhkonline. This is part of solutions at mhkonline. The Aim of this project is to provide a single and standardised page that displays cards, demos or slides. The beauty od this technology is that we don't have to create multiple pages for each slide. Instead we could just pass on the name of the slide to the slide viewer page.

All we had to do was to follow some naming schemes. This project is open source and anyone can use or modify it to suit their needs. A link back or a mention of where you found it would be nice. Also watch this page for future developments on this project.

The Technology

The technology used is javascript. It applies the proper colors for the bg,text,heading etc. It also reads the name of the slide that it should display. Click here to see a demo

The Naming Standards

   We have employed a naming standard for the files.

   * It has 10 characters
   * The First 2 denote the bgcolor of the page
   * the next 2 denote the title color for the page
   * the next 2 denote the text color
   * the next 4 are unique to each card

eg:slideviewer.html?BWRDOR983H

Note! The case of the parameter doesnot matter.
