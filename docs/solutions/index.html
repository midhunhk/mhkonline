Da Play List

This is also a live application that we successfully implemented in our interactive Cds. This project features making a play list using JavaScript and the concept of cookies.  Once the play list is all set and playing starts, you don't have to manually change the song from within the html interface of the interactive cd. After each song ends, the next song in the play list gets played. The aim of this project is to make the interactive interface a completely different and easy one. Unfortunately we couldn't include a preview for 'da Play List' because of space constraints. However the technology will be explained under JavaScript tutorials soon.
