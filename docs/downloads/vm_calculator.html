VM Calculator 3

This is a simple and one of our early projects. It is meant to be an educational purpose and not for any commercial gain. Any feedback or comments will be appreciated. The first big thing that you do when learning a programming language is a calculator program with a number of mathematical operations. We started with C, C++ and now Version 3 in Macromedia Flash. You might want to keep looking at this page in the future in case we publish any bug fixes or better still, another version!

Features

  • This program has two interfaces.
  • The first one enables you to perform the four basic arithmetic operations on two numbers.
  • The second one is a more advanced one offering 12 operations performable on a single number.
  • The first and only calculator program with a background music.
  • Complete with help and documentation.
  • Completely free!
  • Sleek and cool interface.
