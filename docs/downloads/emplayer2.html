eMplayer 2

    It has always been an aim to do a project on something that will be useful to the common people. And this software from Centrum inc Software Solutions is just one step in that direction. The entire development process lasted a couple of months in which we fix one bug, a number of others pop up elsewhere and introduced a number of features. It was quite fun working on this project

    As any project would we were originally only going for perfecting the code. There was little or no graphical enhancements. Then by version 2 we improved graphical interface and by version 2.1 the program became skinnable. Enhanced user experience and documentation is available.

    We have made this project Open Source so that anyone can use it for any educational purposes or other developmental purposes. We have provided the source files free to download. The skin template images for Fireworks are also included.

What this Program can do

  • Has two modes, Quick play and Playlist mode.
  • Can change the volume of the song being played. Since the program cannot interact directly with the hardware, the loudness can be varied between the maximum available to none (mute).
  • Can physically save a playlist to the secondary storage device, so you can easily load the one you wish at the click of a button.
  • This program is skinnable and comes with four skins as default.
  • Shows the name of the currently playing song in it's title.

What this Program can't do

  • It cannot find out the duration of the audio file.
  • You cannot pause at the middle of a song, only stop it altogether.
  • There is no seekbar availability.
