Cryptx

Let us introduce you to CryptX 4.1. It is a fundamentally easy to use encryption/decryption software. The first versions were very simple but did not offer many features, the most prominent of which was the lack of a password protection. To cover all that we bring you the next milestone development from the R & D team at Centrum Inc Software Solutions, Cryptx 4.1. The core functional part of this project is a 32-bit application written in C and compiled using the 32-bit Borland C++ Compiler 5.0.5.

Cryptx 4.1 uses RSA's MD5 Encryption algorithm to protect the password. Since there is no known way to decrypt it, you can believe in its encryption capability completely. It comes with detailed help and documentation. Enhanced graphical presentation and simplicity are the main features of this software.

During the development time, it was called 'Codename Redbox'.
