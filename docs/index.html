Introduction

Welcome to the completely redesigned Mhkonline. This is revision version 3 and better than ever. Starting as just a personal website, this site has grown up to be used as a portal for designers and developers. We provide software based solutions, Tutorials on a number of topics, designing and maintaining websites, cool wallpapers, graphic and poster designing, softwares that we have developed and many more. We have designed this completely from scratch incorporating some features from the last version that we felt were cool to be retained.

We have added a number of exciting features this time with the help of javascript. The most noticable feature is a number of random items that are going on a number of important pages in this site. First of all, you get a random quote below the main navigation bar, a random FMA beside the title, then you get random tutorials in all of the pages that come under 'tutorials.mhkonline'. We have also implemented a small amount of randomness in the wallpapers page. We use an external javascript file to store details about the tutorials and wallpapers. So we only have to update the data in one place, more like using a database. For more info on techincal and design part of this site, go to the design notes in the ae section.

The navigation is easy to use. There are mainly 5 parts to this site and they are cleanly listed in the main navigation bar. Hope you enjoy this site!
